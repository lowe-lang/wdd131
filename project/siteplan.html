Coastal Surf Club

Bringing the joy of surfing to the California coast community

Site Name

Coastal Surf Club

This name represents a club that brings together surfers and surf enthusiasts along the California coast. The name is simple, memorable, and highlights our main focus: surfing.

Optional domain availability: coastal-surfclub.org

Site Purpose

The website provides information about local surf events, club membership, surf lessons, and a gallery of local surf spots. It serves as a hub for the surf community to connect, share tips, and join events.

Scenarios

  • Where can I find the schedule for upcoming surf events?
  • How do I sign up for surf lessons or membership?

Color Schema

Primary color: #1a73e8 — used for headings, buttons, and navigation links
Accent color: #ff7043 — used for highlights, call-to-action buttons, and links

Typography

  • Headings: 'Arial Black', sans-serif
  • Body text: 'Arial', sans-serif
  • Special sections (e.g., quotes or testimonials): 'Georgia', serif

Wireframe

Mobile View:

Header → Navigation → Hero Image → Event Highlights → Membership Call-to-Action → Footer

Desktop/Wide View:

Header (Logo + Nav) → Hero Section with image → 3-Column Event Highlights → Gallery → Membership Section → Footer

CSS

The styling will use the color schema and typography above. CSS will be kept simple and responsive, using flexbox or grid where necessary to support both mobile and desktop views.

Testing

The website will be checked for validation, accessibility, color contrast, SEO best practices, and performance using browser DevTools and Lighthouse.
