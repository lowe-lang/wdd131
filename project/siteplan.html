Bringing the joy of surfing to the California coast community

Site Name

Coastal Surf Club

This name represents a club that brings together surfers and surf enthusiasts along the California coast. It is simple, memorable, and highlights our main focus: surfing.

Optional domain availability: coastal-surfclub.org

Site Purpose

The website provides information about local surf events, club membership, surf lessons, and a gallery of local surf spots. It serves as a hub for the surf community to connect, share tips, and join events.

Scenarios

  • Where can I find the schedule for upcoming surf events?
  • How do I sign up for surf lessons or membership?

Color Schema

Primary color: #1a73e8 — headings, section borders, buttons, and navigation links
Accent color: #ff7043 — highlights, call-to-action buttons, logo border, and wireframe outlines

Typography

  • Headings: 'Arial Black', sans-serif
  • Body text: 'Arial', sans-serif
  • Special sections (e.g., wireframes or callouts): 'Georgia', serif

Wireframe

Mobile View:

Header → Navigation → Hero Image → Event Highlights → Membership Call-to-Action → Footer

Desktop/Wide View:

Header (Logo + Nav) → Hero Section with image → 3-Column Event Highlights → Gallery → Membership Section → Footer

CSS

The styling uses the colors and fonts listed above. CSS is simple, clean, and responsive for both mobile and desktop views using flexbox or grid as needed.

Testing

The website will be tested for HTML/CSS validation, accessibility, color contrast, SEO best practices, and performance using DevTools and Lighthouse.
